BOX MODEL-JOSE MARIA

Se utiliza el box model, para editar el borde de imágenes, párrafos o de cualquier texto en general. Para ello debemos usar los atributo de "width", "height","padding", "border" y "margin".

  • Width y Height: Se usan para cambiar el ancho y alto de la imagen o texto.

  • Padding:Se usa para poner cuanta separación queremos entre el contenido y el margen.

  • Border:Se usa para cambiar el estilo del borde de la imagen.

  • Margin:Se usa para poner cuanta distancia queremos entre cada una de las partes.
