SELECCIÓN POR ETIQUETA

Se puede seleccionar todos los subtítulos, seleccionandolo por la eiqueta "h2", dentro de "style", en "head".

SELECCIÓN POR ID JOSE MARIA RAMIREZ

Se puede seleccionar un texto específico asignándole un nombre o un nick determinado. Apropiando a una etiqueta un id, es decir, h2 id="ejemplo", así de esta forma señalizamos este fragmento y a continuación, usamos # y el nombre asignado "ejemplo", de esta forma tendriamos señalizado el texto. Este párrafo esta coloreado por "id".

SELECCIÓN POR CLASE

Para colorear este párrafo anterior se ha seleccionado por "id". Este párrafo esta coloreado por "id".

USO DEL COMANDO CLASS JOSE MARIA RAMIREZ

Otra forma de seleccionar un texto o atributo específica, es usando la etiqueta "class", la cual actua de la misma manera que la etiqueta "id", salvo que en vez de usar en el "head" # usamos un punto (.) Este párrafo esta coloreado por "class".
